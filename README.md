1. Що таке процес?
Процес — це окрема програма, що виконується в операційній системі та має власний адресний простір і ресурси. У лабораторній роботі процеси використовуються для демонстрації паралельного виконання коду батьківської та дочірньої програм.  

2. Який клас і який простір імен використовується для створення процесу?
Для управління процесами використовується клас Process. Цей клас знаходиться у просторі імен System.Diagnostics.  

3. Як створити і запустити процес?
Для створення процесу необхідно ініціалізувати об'єкт класу Process і вказати шлях до виконуваного файлу у властивості StartInfo.FileName. Запуск здійснюється за допомогою виклику методу Start().  

4. Як зупинити виконання процесу?
Виконання дочірнього процесу зазвичай завершується після виконання всього його коду або очікування введення клавіші користувачем. Також процес можна припинити примусово, закривши вікно консолі або завершивши виконання батьківської програми, якщо процеси пов'язані.  

5. Як визначити завершення процесу?
Про завершення процесу свідчить припинення виводу даних у консоль та автоматичне закриття вікна програми після виконання фінальних команд. Також у коді можна використовувати спеціальні методи очікування завершення, щоб синхронізувати роботу з батьківським процесом.  

6. Що запускає метод Start() класу Process?
Метод Start() запускає виконуваний файл, шлях до якого було вказано у властивостях об'єкта Process. Після цього операційна система створює новий процес, який починає виконувати свій метод Main() паралельно з батьківською програмою.
